# SDK Plugins

ZenWave SDK provides a collection of plugins to generate code from your domain models and API specifications.

## Available Plugins

- [AsyncAPI Generator for Java](/zenwave-sdk/plugins/asyncapi-generator/)
- [AsyncAPI JsonSchema2Pojo](/zenwave-sdk/plugins/asyncapi-jsonschema2pojo/)
- [Avro Schema Compiler](/zenwave-sdk/plugins/avro-schema-compiler/)
- [Backend Application Default](/zenwave-sdk/plugins/backend-application-default/)
- [Kotlin Backend Application](/zenwave-sdk/plugins/customizations/kotlin-backend-application/)
- [OpenAPI Controllers](/zenwave-sdk/plugins/openapi-controllers/)
- [OpenAPI Spring WebTestClient](/zenwave-sdk/plugins/openapi-spring-webtestclient/)
- [OpenAPI Karate](/zenwave-sdk/plugins/openapi-karate/)
- [ZDL to OpenAPI](/zenwave-sdk/plugins/zdl-to-openapi/)
- [ZDL to AsyncAPI](/zenwave-sdk/plugins/zdl-to-asyncapi/)
- [Maven & Gradle Support](/zenwave-sdk/plugins/maven-plugin/)

*Browse the individual plugin documentation pages for detailed configuration options and examples.*
